Тестирование

======================================
1. Зарегистрируйте нового пользователя:
-------------------------------------
curl -H "Content-Type: application/json" -X POST -d '{"username":"test","password":"test"}' http://localhost:8000/register


================================================
2. Войдите в систему (логин и получение токенов):
[--------------------------------------------------------------------------------------------------------
curl -H "Content-Type: application/json" -X POST -d '{"username":"test","password":"test"}' http://localhost:8000/login -c cookies.txt
=================
Просмотр кукисов (опционально, для проверки):
-----------------
cat cookies.txt
---------------------------------------------------]

===========================================
3. Проверка, что текущий токен работает:
[--------------------------------------------------------------------------------------------------------
curl -H "Content-Type: application/json" -X GET http://localhost:8000/protected -b cookies.txt

===================================
4. Обновление токенов:
[-----------------------------------
curl -X POST http://localhost:8000/refresh -b cookies.txt -c cookies.txt
---------
Сделать проверку, что новый токен работает (3)
-------------------------------------------]


===========================
5. Отзыв refresh токена:
[-------------------------------------------------------------------
curl -X DELETE http://localhost:8000/refresh-revoke -b cookies.txt
Проверка обновления токена после отзыва refresh токена (4)
[----------------------------------------------------------
curl -X POST http://localhost:8000/refresh -b cookies.txt
-------------------------------------------------------------------]

===========================
6. Отзыв access токена:
[----------------------------------------------------------------
curl -X DELETE http://localhost:8000/access-revoke -b cookies.txt
---------
Сделать проверку, что текущий токен работает (3)
----------------------------------


===========================
7. Выход из всех остальных сессий:
[--------------------------------------------------------------------------------------------------------
curl -X POST http://localhost:8000/logout-others -b cookies.txt
